Privacy Policy

This Policy explains how the OTG Player Tracker Discord bot (“Bot”) collects and uses information.

1. Data We Receive

  • Command parameters you supply (e.g., platform & username)
  • Discord IDs for the user, channel, and server where the command is invoked (used only for responding)

2. Data We Do Not Store

  • No message content beyond the slash‑command payload
  • No personal Discord profile details (email, phone, etc.)
  • No persistent databases—commands are processed in‑memory and discarded

3. Use of Data

Supplied parameters are forwarded to Tracker.gg solely to build a profile URL or fetch public stats. They are not stored or shared elsewhere.

4. Third‑Party Processing

All stats come from the public API of Tracker.gg. Their servers receive the game username and platform to fulfill the request.

5. Data Retention

No personal data is retained after the command execution completes.

6. Security

  • The Bot runs on private infrastructure with limited access.
  • API keys are stored in environment variables / encrypted secrets.

7. Children’s Privacy

The Bot is intended for general Discord audiences (13+ per Discord ToS). We do not knowingly collect information from anyone under 13.

8. Your Choices

  • Simply stop using the Bot to cease data flow.
  • Server admins can remove the Bot at any time.

9. Changes

Updates to this Policy will be posted at this URL with a new “Last Updated” date.
